1. **Array**: A way to store multiple values under one name.
2. **Function in PHP**: A block of code that does a specific task and can be reused.
3. **One-dimensional array vs Multi-dimensional array**: One stores data in a single row/column, the other in multiple rows/columns.
4. **Global vs Local Variable**: Global accessible everywhere, local only within a function.
5. **Function without vs with parameters**: One doesn't take input, the other does.
6. **Common Predefined Functions**:
   - `strlen()`: Get string length.
   - `strtolower()`: Convert string to lowercase.
   - `strtoupper()`: Convert string to uppercase.
   - `implode()`: Join array elements into a string.
   - `explode()`: Split a string into an array.
7. **Use of include/require**: Useful for including external files in a webpage for code reuse and organization.
8. **Importance of a function**: Organizes code, promotes reuse, makes maintenance easier.
9. **Importance of string functions**: Helps manipulate and process text efficiently.
